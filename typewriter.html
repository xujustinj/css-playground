This clever trick uses background animation to make text appear one character at a time.

It works for multiple lines of text, so long as they are all contained in the same span.

The animation here is at one character per 0.03 seconds.


This section appears after a delay of 7.96 seconds. It is faster, at one character per 0.01 seconds.

By carefully setting the delay, we can make multiple disjoint sections of typewriter text appear in succession.


This only works somewhat with tags. <b> and <i> work great, but underlines (including those on links) do not.

Credit for this trick goes to Temami Afif's amazing article: A Multi-line CSS only Typewriter effect


Unfortunately, all text in the same typewriter block has to be the same colour. Thus, the link above had to be black.

Also, the font must be monospace.
